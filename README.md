# desafio-phishing-dio

Segue a orientação da realização da prática do Desafio de Phishing do curso da DIO:

 1) Acessado o setoolkit no KaliLinux (Maquina estava com rede no modo bridge)
  2) Escolhido o método "Social-Engineering Attacks"
  3) Depois selecionado o Website Attack Vectors
  4) Depois acesso de credenciais e web Templates
  5) Por fim foi selecionado o IP padrão da máquina de teste e passado o url do facebook
  6) Após o usuário fornecer as credenciais no site fake, foi possível analisar elas pelo terminal do kali, como mostrado na imagem anexada.
